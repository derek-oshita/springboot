## SPRING BOOT API

The purpose of this project was to explore the Spring Boot framework. It is a simple backend project that can perform full CRUD on the student object. 

## DEPENDENCIES

- Java 16
- Spring Boot
- Postgres
- Spring JPA
- Maven Plugin
